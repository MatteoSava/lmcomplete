# Changelog

## 0.1.0 - 2026-03-26

Initial public release of `lmcomplete` and `lmc`.

- `lmc "..."` shorthand for `lmc expand "..."`
- `expand`, `explain`, `audit`, `init zsh`, and `stats`
- zsh widget initialization via `lmc init zsh`
- context-aware prompts from shell, cwd, git, and history data
- secret redaction before provider calls
- usage stats persisted on disk
- OpenRouter-backed completions with streaming support
